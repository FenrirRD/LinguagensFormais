Instruções

1 - Selecionar arquivo com o codigo Python
	Clique em "Arquivo" -> Abrir e selecione um arquivo.
2 - Para a analise lexica do arquivo selecionado
	Clique em "Executar" -> Análise Léxica
3 - Gerar arquivo contendo a sequência de tokens, lexemas e número da linha e coluna onde ocorreram
	Clique em "Arquivo" -> Gravar Saída
	O arquivo Saida.lex será gravado no mesmo diretório onde esta o arquivo com o código.
4 - Após executar a análise léxica, é possível executar a análise sintática
	Clique em "Executar" -> Análise Sintática

Estas instruções estão disponíveis também no programa em Ajuda -> Instruções.
A lista de tokens pode ser visualiza em no arquivo ListaTokens.txt que foi enviado junto com o executável.
